## 🚀 Featured Projects

### 🤖 [AI PDF RAG Chatbot](https://github.com/karankavyanjali77-sys/ai-pdf-rag-chatbot)
> *Production-ready document intelligence using Retrieval-Augmented Generation*

A full-stack AI application that allows users to upload multiple PDFs and query them in natural language. Built with a custom RAG pipeline — not a template — with source-page attribution and a clean Streamlit chat interface.

**Stack:** `Python` · `LangChain` · `FAISS` · `Sentence Transformers` · `Groq LLM` · `Streamlit` · `PyPDF`

**Highlights:**
- 📄 Multi-PDF ingestion and semantic chunking
- 🔍 FAISS vector store for fast similarity search
- 💬 Context-aware Q&A with source-page citation
- ⚡ Groq LLM integration for low-latency inference
- 🖥️ Deployed chat-style UI via Streamlit

---

### 📊 [Customer Segmentation with ML](https://github.com/karankavyanjali77-sys/customer-segmentation-ml)
> *Unsupervised learning applied to real-world business intelligence*

An end-to-end ML analysis pipeline using clustering algorithms to extract actionable customer segments from transaction data — the kind of insight businesses actually pay for.

**Stack:** `Python` · `scikit-learn` · `Pandas` · `Matplotlib` · `Jupyter`

**Highlights:**
- 🧮 K-Means / hierarchical clustering with optimal cluster selection
- 📈 Visualised segment profiles for business interpretation
- 🧹 Full EDA and feature engineering pipeline
